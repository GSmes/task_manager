# Task Manager
__Created by: Garrett Smestad__<br><br>
## Project Overview
Task Manager is an application that is more or less a to-do list application. It allows the user to add tasks, using a title and description for each task, to a list, which is displayed on the page http://localhost:9393/tasks.
